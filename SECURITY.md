# Security Policy

| Version | Supported          |

## Reporting a Vulnerability

If you find a vulnerability within the project's code or documentation (e.g., an accidental inclusion of unsafe code), please report it responsibly.

**Do not open a public issue.** Instead, please contact the maintainer directly at [INSERT EMAIL ADDRESS].

We follow coordinated disclosure and will work to address any issues promptly.

## Scope of Research

This project is for research and education. Researchers using the information here are expected to:

- Act ethically.
- Respect privacy.
- Operate only on systems they own or have explicit permission to test.
