# Glossary of Terms

| Term | Definition |
| :--- | :--- |
| **HID** | **Human Interface Device.** A USB class for devices that interact directly with humans (Keyboard, Mouse, Gamepads). |
| **Composite Device** | A single physical USB device that presents multiple logical interfaces (e.g., a Keyboard and a Serial port simultaneously). |
| **Enumeration** | The process where the Host (PC) identifies and configures a newly connected USB device. |
| **VID / PID** | **Vendor ID / Product ID.** Hexadecimal identifiers used by the OS to identify the manufacturer and model of a USB device. |
| **BadUSB** | A general term for any USB device that has been reprogrammed to perform malicious actions, typically via HID emulation. |
| **Teensyduino** | The software extension that enables the Arduino IDE to program Teensy microcontrollers. |
| **RNDIS / NCM** | USB classes for Ethernet-over-USB, allowing a device to appear as a network adapter. |
| **Payload** | In this context, the Arduino/C++ code loaded onto the Teensy to execute a specific sequence of actions. |
